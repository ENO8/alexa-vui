About VUI

Today, people spend too much of their time looking for information. Visualizations provide the insights needed to foster a data-driven culture, but sometimes all that information can be intimidating.

VUI can help solve this problem by acting as an intelligent assistant at work. VUI lets people use their voice to interact with technology so they can spontaneously ask questions in a way that feels natural.

Tech Summary

AwS Alexa kit to operate voice commands
AWS Lambda function to manipulate command
Web Socket connectivity for real time response at website
angular 6 is used at the website
